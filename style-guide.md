# Front-end Style Guide

## Layout
boxes 
largura 351px
altura 257px ou 250px
padding 30px
bordas coloridas 
altura 4px
margem da caixa ao texto 60px;
espaco entre o texto do h1 22px
espaco entre o titulo e o paragrafo 30px

The designs were created to the following widths:

- Mobile: 375px
- Desktop: 1440px

## Colors

### Primary

- Red: hsl(0, 78%, 62%)
- Cyan: hsl(180, 62%, 55%)
- Orange: hsl(34, 97%, 64%)
- Blue: hsl(212, 86%, 64%)

### Neutral

- Very Dark Blue: hsl(234, 12%, 34%)
- Grayish Blue: hsl(229, 6%, 66%)
- Very Light Gray: hsl(0, 0%, 98%)

## Typography

### Body Copy

- Font size: 15px
